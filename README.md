# 🔒 CryptTitan: Advanced Web-Based File Security

Welcome to **CryptTitan**! 🚀  
This is a robust web application designed to simplify file security for everyday users. By combining encryption, decryption, and compression into a single browser-based interface, **CryptTitan** ensures your data remains private and manageable without needing complex terminal commands.

---

## ✨ Core Features

* **Titan-Grade Encryption (AES):** Secure your sensitive documents using the industry-standard AES algorithm.
* **Seamless Decryption:** Effortlessly restore your encrypted files to their original state with a few clicks.
* **Integrated Compression:** Shrink file sizes on-the-fly for faster sharing and efficient storage.
* **Universal Web GUI:** A clean, modern, and intuitive interface that brings high-level security to the browser.
* **Secure File Handling:** Built-in upload and download system designed for a smooth user experience.

## 🛠️ Tech Stack

This project leverages a modern JavaScript-centric ecosystem:

* **Backend:** [Node.js](https://nodejs.org/) & [Express.js](https://expressjs.com/) (Managing the server-side logic).
* **Security:** [Crypto-JS](https://www.npmjs.com/package/crypto-js) / AES Libraries (Powering the cryptographic engine).
* **File Processing:** [Multer](https://www.npmjs.com/package/multer) (Upload management) & [Archiver](https://www.npmjs.com/package/archiver) (Compression logic).
* **Frontend:** HTML5, CSS3, and Vanilla JavaScript for a fast and responsive UI.

## 🚀 Installation & Setup

Want to deploy **CryptTitan** locally? Follow these steps:

1.  **Prerequisites:** Ensure you have [Node.js](https://nodejs.org/) installed on your machine.
2.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/dianagustinee/CryptTitan.git](https://github.com/dianagustinee/CryptTitan.git)
    ```
3.  **Install Dependencies:**
    Navigate to the project folder and run:
    ```bash
    npm install
    ```
4.  **Launch the App:**
    Start the server with:
    ```bash
    node app.js
    ```
5.  **Access in Browser:**
    Open your browser and navigate to: `http://localhost:3000`
