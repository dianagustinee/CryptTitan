Turn any file into unreadable ciphertext with a passphrase only you know.

Encryption is the process of scrambling readable information into a secret code so no one else can open it without your key. Upload a file, set a strong password, and CrypTitan produces a locked encrypted_ copy powered by AES-256-CBC. Every operation is stamped into your activity log below.

  • в–ё AES-256-CBC with SHA-256 derived key
  • в–ё Random 16-byte IV per file
  • в–ё Activity log kept locally

// DECRYPT

Restore your original file from an encrypted_ copy вҖ” instantly.

Decryption reverses the scramble. Provide the encrypted file and the exact same password you used to lock it. CrypTitan verifies the payload, applies the AES-256 decipher, and writes back a clean decrypted_ copy. Wrong key = access denied, so keep your passphrase safe.

  • в–ё Works only on encrypted_* files
  • в–ё Password must match the original
  • в–ё Recovery log auto-saved

Shrink files down using lossless GZIP вҖ” smaller size, same data.
